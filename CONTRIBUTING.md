# Contributing to Signatury

Signatury is currently maintained by a single author.

## Maintenance Model

- The repository is public so the code can be used freely.
- Forks and private adaptations are welcome under the terms of the `MIT` license.
- External code contributions are not the primary workflow for this project at the moment.

## Feedback

If you find a bug or want to suggest an improvement, opening an issue is the best path when issue tracking is enabled.

## Development Principles

- Keep Windows PowerShell 5.1 compatibility by default.
- Avoid third-party PowerShell modules unless strictly necessary.
- Keep the project usable in offline, on-prem Active Directory environments.
- Do not include proprietary templates, logos, secrets, customer data, logs, cache files, or temporary output.

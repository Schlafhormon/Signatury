# Templates

This folder contains publicly safe example templates for Signatury.

## Included Sample

- `DK_Standard.docx`: neutral sample template for local testing

## Notes

- Replace the sample with your own corporate template when deploying Signatury in production.
- Keep the placeholder names exactly as required by the configuration unless you update `VariableMappings`.
- Avoid committing proprietary logos, internal contact details, or customer-specific assets to a public repository.
